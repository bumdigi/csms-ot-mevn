# csms-ot-mevn project

MEVN Stack 은 Mongo DB + Express.js + Vue.js + Node.js 로 이루어진 프로젝트를 가리킴
프로젝트 시작전에 mongo DB를 설치하고 localhost 하위에 csms-ot 라는 인스턴스를 생성해야 함
'mongodb://localhost:27017/csms-ot'

# Project setup
```
npm install
```

### 실행
```
cd csms-ot-mevn 
```

```
cd npm run server
```
